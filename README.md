# Git Timewarp
Git Timewarp is a project that aims to provide a unique solution for analyzing and visualizing Git repository data. The project utilizes a combination of natural language processing and machine learning techniques to provide insights into the repository's history and structure. With Git Timewarp, developers can gain a deeper understanding of their repository's evolution and identify trends and patterns that can inform future development decisions.

## Tech Stack
The project is built using a Python-based tech stack, with the following key technologies:
* Python 3.x
* YAML configuration files
* JSON data storage

## Project Structure
The project is organized into the following main directories and files:
* `git_timewarp`: The main project directory, containing:
	+ `main.py`: The primary entry point for the application
	+ `config.yaml`: Configuration file for the application
	+ `embedding_cache.json`: Cache file for embedding data
	+ `requirements.txt`: List of dependencies required by the application
	+ `render.yaml`: Configuration file for rendering
	+ `phase2_guide.txt`: Guide for phase 2 of the project

## Getting Started
To get started with the project, follow these steps:
1. Clone the repository using `git clone`
2. Navigate to the `git_timewarp` directory
3. Install the required dependencies using `pip install -r requirements.txt`
4. Run the application using `python main.py`
5. Refer to the `config.yaml` and `render.yaml` files for configuration options and rendering settings.
